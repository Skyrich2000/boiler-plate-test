# 따라하면서 배우는 노드 리액스 강의 노트

## 1강

* npm init
* npm install express --save
* 공홈가서 기초 코드 복사
* "scripts": { "start": "node index.js" }  추가
* npm run start

## 2강

* 몽고 db 회원가입
* 몽고 db 공홈에서 클러스터 만들기
  * 클라우드 상에서 만드는것
  * 싱가포르 프리 티어
  * 이름은 맘대로
* 유저 만들기
  * CONNECT 눌러서 유저 생성
  * connect your application
  * 연결 코드 복사
* npm install mongoose --save
* 몽고 db 연결 코드 작성
  * password 랑 dbname 잘 치환
  * use~ 머시기들 쓰고
  * then, catch로 확인
